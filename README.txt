# QA-Automation-Demo

This Python mini-project demonstrates QA automation logic for AI agent states. It includes:

- `main.py`: Sandbox for testing AI agent state logic
- `log_checker.py`: Validates AI agent states in logs
- `ai_pipeline_demo.py`: A simple pipeline simulating AI-driven recovery suggestions

## Purpose
Showcase automation, validation, and pipeline thinking relevant to QA/DevTest roles.
